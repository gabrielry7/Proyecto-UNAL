# HP2C Streamlit Dashboard

Dashboard financiero para el proyecto HP2C: arepas congeladas high-protein para exportar de Colombia a Estados Unidos.

## Cómo correrlo

1. Instala las dependencias:

```bash
pip install -r requirements.txt
```

2. Corre la app:

```bash
streamlit run streamlit_app.py
```

## Qué incluye

- Selector de escenario: Base, Optimista y Pesimista.
- KPIs financieros.
- Gráficas de ingresos, costos, EBITDA, paquetes vendidos y márgenes.
- Comparación de escenarios.
- Tabla financiera completa.
- Valoración pre-money, post-money y participación sugerida para una inversión de US$90.000.
